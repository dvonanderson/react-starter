# react-starter

###Installation Instructions

- Clone repository
```
git clone git@github.com:dsawebdev/react-starter.git
```
- CD into Directory 
```
cd react-starter <or whatever you named the clone repository> 
```
- Install dependecies
```
yarn
```
- Start local server
```
yarn start
```
- Open up your prefered web-brower and view the app:
```
localhost:3000
```
